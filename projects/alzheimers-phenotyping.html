Research Project

Multimodal Alzheimer’s Phenotyping

A translational rodent neuroscience project integrating behavioral motor learning, electrophysiology, and molecular validation for Alzheimer’s disease mechanisms.

Active Individual Project Behavior EEG Molecular Validation

Overview

This project investigates motor skill learning and implicit memory impairments in amyloid-beta injected rodent models of Alzheimer’s disease. By combining behavioral tasks, electrophysiology acquisition, and post-mortem validation, the project aims to provide reproducible biomarkers of cognitive and motor decline.

Scientific Objectives

The main objectives include (1) quantifying behavioral motor learning deficits, (2) extracting EEG-based signatures of neurodegeneration, (3) validating findings using molecular and histological workflows, and (4) developing standardized analysis pipelines for reproducible research reporting.

Methods & Workflow

The workflow includes animal training paradigms, motor skill learning behavioral assays, synchronized electrophysiology acquisition, signal preprocessing and QC, statistical modeling, and experimental validation using tissue processing and molecular assays.

Expected Outputs

Expected outputs include publishable datasets, reproducible behavioral + EEG pipelines, standardized reporting templates, and open-source software modules for multimodal neuroscience analysis.

Collaboration Opportunities

We welcome collaborations with neuroscience laboratories interested in behavioral phenotyping, electrophysiology analysis, Alzheimer’s disease modeling, and multimodal experimental validation.
